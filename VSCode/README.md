# How to set up VSCode in Ubuntu EC2 server

## Set up procedure

1. Downloading VSCode debian file
   ```bash
   curl -L https://go.microsoft.com/fwlink/?LinkID=760868 -o vscode.deb
   ```

2. Installing VSCode
   ```bash
   sudo apt install ./vscode.deb
   ```
